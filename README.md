# Harikrushn Agro Chemicals

A modern React + TypeScript website for Harikrushn Agro Chemicals, a trusted agricultural products dealer in Lathidad, Gujarat.

## Features

- Responsive design for all devices
- Product catalog with enquiry forms
- Gallery showcase
- WhatsApp integration for quick enquiries
- Beautiful animations with Framer Motion
- SEO optimized

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- Framer Motion
- React Router DOM
- React Hook Form + Zod

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Run linter
npm run lint
```
